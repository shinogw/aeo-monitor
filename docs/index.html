自社と競合の言及率を並べて比較。業界内でのポジションと目指すべき水準が明確に。

自社管理で安全

外部サービスにデータを預ける必要なし。自分の環境で実行するので、機密情報も安全。

使い方（3ステップ）

1

設定ファイルを作成

業種、自社ブランド、競合、チェックしたい質問をJSONで定義。テンプレートをそのまま編集するだけ。

2

モニタリング実行

コマンド一発でAI検索エンジンに質問を投げ、ブランドの言及率を統計的に計測。

3

ダッシュボードで確認

結果はWebダッシュボードで可視化。改善提案も自動生成。クライアントへの報告にそのまま使えます。

# 1. ダウンロード
git clone https://github.com/shinogw/aeo-monitor.git
cd aeo-monitor

# 2. 設定ファイルを編集
cp configs/sample-tax.json configs/my-client.json
vim configs/my-client.json

# 3. モニタリング実行
export OPENAI_API_KEY="sk-..."
python3 aeo_monitor.py configs/my-client.json

# 4. ダッシュボード起動
python3 server.py
# → http://localhost:8080
