# AEO対策方法 — 業種別実践ガイド

## AIが店舗・サービスを推薦する仕組み

AIは以下のソースから情報を集めて推薦を決める：

| AI検索エンジン | 主な情報ソース |
|---|---|
| **ChatGPT** | Bing検索インデックス、食べログ、TripAdvisor、個人ブログ、Reddit |
| **Gemini** | Google検索全体、Googleビジネスプロフィール、Google Maps口コミ |
| **Perplexity** | 独自クローラー+Bing、権威性の高いメディア、レビューサイト |

**日本語クエリ**: 食べログが圧倒的に引用される
**英語クエリ**: TripAdvisorが最重要
**共通**: Google Maps口コミは全AIがベースラインとして参照

## AEO対策 5つの柱

### 1. 🔄 鮮度（最重要）
**問題**: 古い情報はAIに無視される（95%の引用が10ヶ月以内のコンテンツ）

**対策**:
- 公式サイトのメニュー・営業時間を**月1回以上**更新する
- 季節メニュー・イベント情報を発信する
- ページに「最終更新日」を表示する（引用率1.8倍）
- Googleビジネスプロフィールに**週1回以上**投稿する

**効果**: 定期更新だけでAI引用率が41%向上（AirOps調査）

---

### 2. 📋 構造化（Schema.orgマークアップ）
**問題**: AIが情報を「読み取れない」形式だと引用されない

**対策**:
- 公式サイトにSchema.org `Restaurant` / `LocalBusiness` マークアップを設置
- FAQページを作る（「予約は必要？」「駐車場はある？」「アレルギー対応は？」）
- メニューを箇条書き・リスト形式で掲載（長文パラグラフは引用率59%低下）

**効果**: FAQ/HowToスキーマ使用で引用率78%向上

---

### 3. ⭐ 権威性（口コミ・メディア掲載）
**問題**: 口コミが少ない・メディア掲載がないとAIが「確信を持って推薦できない」

**対策**:
- 口コミを**最低50件以上**に（20件未満は47%の消費者が利用回避）
- 食べログ・Google Maps・TripAdvisorの口コミを**全て管理**
- ネガティブレビューに**24時間以内に丁寧に返信**
- メディア掲載実績を公式サイトに掲載
- シェフ・オーナーのプロフィール・経歴を明記

**効果**: 外部引用が500語あたり2件以上あると引用数2.4倍

---

### 4. 📍 ローカル最適化（エリア特化）
**問題**: 「〇〇駅近く」「〇〇周辺」のクエリでAIが推薦する情報がない

**対策**:
- Googleビジネスプロフィールの説明文に**ランドマーク名**を入れる（「コマ劇場から徒歩2分」）
- 公式サイトに詳細なアクセスページ（各駅からの徒歩ルート）
- 全プラットフォームで**店名・住所・電話番号を完全一致**させる（NAP一貫性）
- 対象エリアのキーワードをH2/H3見出しに含める

**効果**: GBP未設定の店舗は65%（BrightLocal）→ 設定するだけで差別化

---

### 5. 🗣️ ブランド言及（ネット上の存在感）
**問題**: ネット上で店名が言及されていないとAIが知らない

**対策**:
- 食べログ、ぐるなび、ホットペッパー、Retty、TripAdvisorの**全てに登録・最新化**
- フードブロガー・インフルエンサーに取材依頼
- 「〇〇エリアのおすすめ」系記事に掲載されるようPR
- 来店客にSNS投稿を促す（ハッシュタグ統一）

**効果**: 複数プラットフォームでの言及がAI推薦の基盤
